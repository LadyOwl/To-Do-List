# To-Do List — консольный менеджер задач

Консольное приложение на Python для управления списком задач. Позволяет добавлять, редактировать, удалять и отмечать задачи как выполненные. Реализовано в процедурном стиле с использованием встроенных структур данных.

## 🚀 Функциональность

- Добавление новой задачи
- Просмотр всех задач с нумерацией
- Редактирование описания существующей задачи
- Удаление задачи по номеру
- Отметка задачи как выполненной / снятие отметки
- Выход из программы с подтверждением
- Обработка некорректного ввода (пустой ввод, несуществующий номер задачи)

## 🛠 Стек

- **Python 3.x**
- Стандартная библиотека (без внешних зависимостей)
